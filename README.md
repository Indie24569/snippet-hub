# SnippetHub

SnippetHub is a web-based snippet management platform built using React, TypeScript, and Vite. It allows users to create, organize, pin, edit, and manage text, code, and link snippets through a clean and intuitive interface.

The project was developed as a solo frontend application to practice component-based architecture, TypeScript integration, dynamic UI rendering, and state management in modern React applications.


## Features

- Create and manage snippets
- Support for text, code, and link snippets
- Organize snippets into categories
- Pin important snippets
- Edit and delete existing snippets
- Dynamic sidebar category management
- Modal-based snippet creation workflow
- Persistent data storage
- Responsive and clean UI


## Tech Stack

- React
- TypeScript
- Vite
- CSS
- JavaScript
